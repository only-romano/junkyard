### _HINTS_

Ordered list `<ol>` automatically add numbers as bullet points for `<li>` (before `<li>` list items).

Use to create a numbered list when order is important!

```html
<ol>
    <li>Gold Medalist</li>
    <li>Silver Medalist</li>
    <li>Bronze Medalist</li>
</ol>
```

Results in:

1. Gold Medalist
2. Silver Medalist
3. Bronze Medalist
